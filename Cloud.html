Latitude vs. Cloud Cover

Surprisingly, there does not appear to be much coorelation between the latitude of a given city and it's respective cloud cover. There is a rather large cluster of cities with 0%, or close to 0%, cloud cover and a two rather prominent groups of cities near the 75% and 90% levels.

It is also interesting to note that there is obvious horizontal grouping of cities. I hypothesize this to be because cloud cover is being approximated, perhaps to the nearest 5%.
